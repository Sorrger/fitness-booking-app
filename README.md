Fitness Booking App
Aplikacja dla trenerów i osób ćwiczących, umożliwiająca rezerwację treningów i zarządzanie harmonogramem zajęć.


Technologie:
<ul>
  <li>Frontend: (HTML, CSS, Bootstrap) – zapewniający responsywny i nowoczesny interfejs użytkownika.</li>
  <li>Backend:  – lekka i szybka technologia do obsługi żądań oraz przetwarzania danych.</li>
  <li>Baza danych:  – przechowująca informacje o użytkownikach, rezerwacjach i dostępnych treningach.</li>
</ul>
Technologie:
<ul>
  <li>Rejestracja i logowanie użytkowników oraz trenerów</li>
  <li>Możliwość rezerwacji/tworzenia zajęć</li>
  <li>Integracja z bazą  do przechowywania danych</li>
  <li>Dostępne opinie w profilach od Trenerów</li>
</ul>
Instrukcja uruchomienia lokalnego:
<ul>
  <li>git clone https://github.com/twoja-nazwa-uzytkownika/fitness-booking-app</li>
  <li>cd IO-activitysite</li>
  <li>python -m venv venv</li>
  <li>venv\Scripts\activate</li>
  <li>pip install -r requirements.txt </li>
  <li>uvicorn main:app --reload   </li>
</ul>
